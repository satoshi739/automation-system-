# 本番データ投入ヒアリングシート

このファイルに答えを書いてもらえれば、そのままYAMLに変換します。
空欄・「不明」・「後で」でも問題ありません。

---

## 【A】クライアント一覧

**現在進行中のクライアントを教えてください。**
1社ずつ以下の形式で答えてください。

```
クライアント名:
担当ブランド(dsc-marketing / cashflowsupport / upjapan / satoshi):
担当者名:
担当者メール:
担当者連絡先(Slack/LINE等):
```

---

## 【B】各案件の詳細

**【A】で挙げたクライアントそれぞれについて答えてください。**

```
──────────────────────────────
クライアント名:

  【案件種別】
  月額 or 単発:
  案件名(例: ◯◯社 SNS運用支援):
  業務内容を一言で:

  【金額】
  月額費用(税込): ¥           ← 月額の場合
  単発総額(税込): ¥           ← 単発の場合

  【スケジュール】
  開始日: YYYY-MM-DD
  終了予定日: YYYY-MM-DD      ← 月額なら空欄でOK

  【今の状態】
  フェーズ(P1〜P5): P
    P1=リード / P2=提案中 / P3=契約交渉 / P4=進行中 / P5=完了/継続

  【請求状況】
  最終請求書送付日: YYYY-MM-DD
  最終入金確認日:  YYYY-MM-DD
  未払いはあるか(はい/いいえ):

  【次のマイルストーン】
  マイルストーン名:
  期日: YYYY-MM-DD

  【納品物】
  (箇条書きで)
  -
  -

  【気になること・懸念点】(なければ空欄)
──────────────────────────────
```

---

## 【C】顧客の健康状態

**各クライアントについて直感で答えてください。**

```
クライアント名:
  満足度 (high/neutral/low/unknown):
  リスクはあるか (はい/いいえ):
  リスクがある場合、どんなシグナル?
    例: 返信が遅い / 効果に不満 / 予算削減の話が出た
  次にコンタクトする予定日: YYYY-MM-DD
  今月実施したアクション:
    例: 5/1 月次定例MTG
```

---

## 【D】財務の現状（今月分）

```
今月(2026-05)の状況:

  MRR（月初時点の月額売上合計）: ¥
  今月 新規契約したMRR: ¥        ← なければ 0
  今月 解約になったMRR: ¥        ← なければ 0

  今月の単発売上（入金済み）: ¥   ← なければ 0

  外注費・ツール費など直接原価: ¥ ← なければ 0

  今月 発行した請求書: __件
  うち 入金済み: __件
  うち 未払い(期日超過): __件
  未払い金額合計: ¥

  ブランド別のMRR内訳:
    dsc-marketing:     ¥
    cashflowsupport:   ¥
    upjapan:           ¥
    satoshi:           ¥

  今月 新規契約したクライアント名・MRR:
    例: 株式会社◯◯ ¥80,000
  今月 解約したクライアント名・理由:
    例: (なし)
```

---

## 【E】前月（2026-04）の実績も分かれば

```
  4月末時点のMRR: ¥
  4月の単発売上: ¥
  4月の直接原価: ¥
  4月の入金未完了はあったか(はい/いいえ):
```

前月データがあると「先月比 +¥◯◯」の表示が正確になります。
