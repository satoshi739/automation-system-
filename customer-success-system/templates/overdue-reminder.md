# 未回収請求 督促テンプレート

scheduler の `overdue_alert_job` がLINEに通知する。
本ファイルは **人間が督促文を送る際の文例集**。

---

## 段階1: 14日経過（柔らかめ・連絡漏れ確認）

件名: 【ご請求のご確認】請求書 No.{INVOICE_NO} のご入金状況

```
{CLIENT_NAME}
ご担当者様

いつも大変お世話になっております。
{YOUR_NAME}でございます。

{INVOICE_DATE}付でお送りした請求書 No.{INVOICE_NO}（金額: ¥{AMOUNT:,}）について、
本日時点でご入金の確認が取れておらず、ご連絡いたしました。

行き違いになっておりましたら大変失礼いたしました。
ご都合のよろしいタイミングでご確認いただけますと幸いです。

ご不明点や請求書の再送が必要な場合は、本メールにご返信ください。

何卒よろしくお願い申し上げます。
```

---

## 段階2: 30日経過（明確に督促・期日提示）

件名: 【ご入金のお願い】請求書 No.{INVOICE_NO}（{DAYS_OVERDUE}日経過）

```
{CLIENT_NAME}
ご担当者様

お世話になっております。{YOUR_NAME}です。

{INVOICE_DATE}付でお送りした下記請求書について、
お支払期日を {DAYS_OVERDUE} 日経過しております。

────────
請求書: No.{INVOICE_NO}
金額: ¥{AMOUNT:,}（税込）
請求日: {INVOICE_DATE}
お支払期日: {DUE_DATE}
────────

恐れ入りますが、{NEW_DUE_DATE} までにお振込みいただけますよう、
お願い申し上げます。

ご事情等ございましたら、お早めにご相談ください。
ご対応のほど、よろしくお願いいたします。
```

---

## 段階3: 45日以上経過（最終通知）

件名: 【最終ご通知】請求書 No.{INVOICE_NO} ご入金のお願い

```
{CLIENT_NAME}
ご担当者様

お世話になっております。{YOUR_NAME}です。

本件、複数回ご連絡しておりますが、ご入金の確認が取れておりません。

────────
請求書: No.{INVOICE_NO}
金額: ¥{AMOUNT:,}（税込）
請求日: {INVOICE_DATE}（{DAYS_OVERDUE}日経過）
────────

誠に恐縮ですが、{FINAL_DUE_DATE} までにお振込みなき場合、
弊社規定に基づき、サービス提供の停止および法的手続きの
検討を進めさせていただく旨、ご了承願います。

ご事情等あれば、必ずご一報いただきたく存じます。
何卒よろしくお願い申し上げます。
```

## 運用ルール

| 経過日数 | アクション | 送信媒体 |
|---------|-----------|---------|
| 7日 | 自動LINE通知（Satoshi宛・本人確認） | LINE |
| 14日 | 段階1テンプレ送信 | Email |
| 21日 | 電話/Chatで一次確認 | 電話 |
| 30日 | 段階2テンプレ送信 | Email |
| 45日 | 段階3テンプレ送信 | Email + 内容証明検討 |
| 60日 | 法的措置の検討着手 | — |

## 現在の対象

（該当なし）

未収請求書は `finance-system/logs/<YYYY-MM>.yaml` の `overdue_invoices` から自動抽出される想定。
